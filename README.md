# morganfung.com

My personal site. React + Vite, deployed to GitHub Pages at [morganfung.com](https://www.morganfung.com).

## Develop

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

Built output goes to `dist/`. Static assets (favicon, OG image, `CNAME`, `404.html` SPA fallback) live in `public/`.
